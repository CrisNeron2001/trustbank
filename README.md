# Trust bank
-------------

## Project description

The main purpose of this project is to present a simulated banking system built using a microservices architecture with Spring Boot. It is designed to demonstrate how core banking domains such as user management, accounts, and transactions can be separated into independent services, each with its own responsibility and database.
